Analyze the provided project structure and generate a comprehensive directory tree visualization.

## Format

Use the following tree structure:

```tree
project-name/                                  # Brief project description
├── directory-name/                            # Directory purpose
│   ├── file.ext                               # File description
│   ├── subdirectory/                          # Subdirectory purpose
│   │   ├── component.ext                      # Component description
│   │   └── component.ext                      # Component description
│   └── config-file.ext                        # Configuration purpose
└── root-level-file.ext                        # File description
```

## Requirements

- 3-4 levels of depth for main directories
- Concise one-line descriptions for each element
- Include all major directories and key files
- Omit irrelevant dotfiles and cache directories
- Keep descriptions informative but brief

## Output

Generate only the tree structure. No additional text or sections.
