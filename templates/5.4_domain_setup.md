```typescript

```

**Backend Request DTO (snake_case):**

```python

  user_name: str
  access_token: str
```

**Backend Response DTO (snake_case):**

```python
class {{ResponseName}}DTO:
  user_id: int
  session_token: str
```

**Frontend Response DTO (camelCase):**

```typescript
interface {{ResponseName}}DTO {
  userId: number;
  sessionToken: string;
}
```
